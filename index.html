 Nest - CurbSide.ai   An Interactive Mapping Platform and Data Dashboard built for real-time actionable city planning
Nest - CurbSide.ai

Nest is CurbSide's mapping platform built for private companies and cities to manage their fleet. The platform works with CurbSide's BirdsEye Unit to aggregate data and provides visualizations over a map to understand that data. What sets this apart from other platforms is the user's ability to take decisions and implement those using Nest's drawable interface which allows real-time planning and execution performed over the fleet via BirdsEye.

I have been building Nest for a year now with continuous support from the City of Atlanta Department of Transportation. Even though the platform can be accessed only via invitation, you can find more about it here: curbside.ai/tech

 Alexa Finesse - HackGT 6   Alexa Skill that helps you manage your finances | Best use of NCR's Banking API | Best Cloud Computing Hack
Alexa Finesse - built for HackGT 6

Finesse is an Alexa Skill that helps college students manage their funds using a voice user interface. It works on NCR's banking API and uses NLP to categorize transactions for a better budgeting.

This project won two awards at HackGT 6 - Best use of NCR's banking API and Best Cloud Computing Hack for the FinTech industry by Accenture! It can be found here on Devpost

Some of the ways you can talk to Finesse on any Alexa enabled device are:

• Alexa, open Finesse (Requires AT&T Authentication)
• What is my current balance? (NCR API)
• Analyze my expenditure (NCR API)
• Give me financial advice (NCR API + NLP)
• Find me some student deals (BeautifulSoup Web Scraping)
• Define "Savings Account" (Merriam-Webster API)
• Set my monthly budget to $300
• How is Apple performing? (Blackrock + Yahoo API)

 eS2 - EyeScribe  EyeScribe is a spectacle based assistive device for the visually impaired built on a Pi and uses Handwritten Text Recognition
eS2 - EyeScribe

EyeScribe is an assistive device for the visually impaired that uses computer vision to recognize digital and handwritten text and then uses text-to-speech to provide audio dictation of the text. eS2 is the second version which I am working on right now. The previous version has been used by a couple of schools for the blind children in Delhi, India.

eS2 can be used in everyday life because it uses OpenCV's EAST algorithm to detect text in a noisy and unstabilised environment.

EyeScribe won Prudential Financial's Annual Spirit Community Award from India and I was awarded a Gold Medallion by Michael Phelps in Washington D.C.

OpenCV | Google Tesseract | eSpeak | Raspberry Pi | Python | Linux

 Decibel - HackGT 5  A web app which helps you learn a new language with a voice-based pronunciation feedback
Decibel - HackGT 5

Decibel is a program that helps users work on their pronunciation while learning a new language by letting them record a sentence and compare it with a Google Text-to-Speech recording of the same sentence to get a feedback. It is an audio analysis program that shows a random sentence in the target language and requests an input audio from the user. It extracts its Mel-Frequency Cepstral (MFC) coefficient and matches it to that of a Google Cloud TTS audio output of the same sentence. We use the process of Dynamic Time Warping to achieve this comparison between the two temporal sequences. The matching produces a numerical value which we present as the percentage similarity between the human-generated and the machine-generated audio files.

It can be found here on Devpost

Python | Google TTS | Google Cloud | JavaScript | Flask

 Carecino  A CNN based image diagnosis app for actinic keratosis. Finalist at Intel IRIS Fair and INSEF National Fair
Carecino

Carecino is a research and development project which can be classified into diagnosis and treatment of skin cancer/precancer with a particular focus on actinic keratosis. Diagnosis involves a deep learning approach using a 8-way classifier. Deep CNN is built on Inception-v3 by retraining its final layer using TensorFlow. I used ISIC Archive's lesion images as the training and validation set.

I did this research under Dr Jaya Shukla (UICC Fellow), connected by Indian Council of Medical Research, New Delhi.
This project made it to the final round and was showcased at Intel's IRIS National Fair in Delhi.

Python | TensorFlow | Deep Learning | Image Classification | Linux
